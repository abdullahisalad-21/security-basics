# Security Fundamentals for IT Support

This repository contains my security learning and hands-on practice. It covers essential concepts used in real IT environments, including password policies, ACLs, least privilege, and system hardening.

## Skills Demonstrated
- Password policy configuration
- Access Control Lists (ACL, DACL, SACL, ACE)
- Least privilege implementation
- System hardening techniques
- Security awareness and customer-facing security skills

This repo shows my understanding of security principles required for IT Support, SysAdmin, and cybersecurity pathways.
